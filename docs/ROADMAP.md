# Roadmap

Future features and improvements planned for Dinder:

## Social Features
- [ ] Follow other users and see their recipes
- [ ] Like and comment on recipes
- [ ] Share recipes to social media platforms
- [ ] User profiles with recipe collections

---

## Recipe Enhancements
- [ ] Recipe reviews and star ratings
- [ ] Nutrition information display (calories, macros)
- [ ] Advanced search with full-text capabilities
- [ ] Recipe collections and categories
- [ ] Recipe duplication/forking

---

## Meal Planning
- [ ] Shopping list generation from weekplans
- [ ] Grocery store integration for delivery
- [ ] Meal prep suggestions and batch cooking
- [ ] Calorie and macro tracking

---

## Mobile Experience
- [ ] Native mobile app (React Native)
- [ ] Offline mode with local storage
- [ ] Push notifications for meal reminders

---

## AI & Automation
- [ ] AI-powered recipe recommendations
- [ ] Ingredient substitution suggestions
- [ ] Automated meal plan generation based on preferences

---

## Performance & Technical
- [ ] Lazy loading for images and routes
- [ ] Service worker for offline caching
- [ ] Bundle size optimization and code splitting
- [ ] Database query optimization and indexing
- [ ] Progressive Web App (PWA) support
- [ ] Error tracking and logging system
- [ ] API response caching strategies
- [ ] Extensive Testing

---

## Other
- [ ] Recipe import from URLs
- [ ] Print-friendly recipe cards
- [ ] Multi-language support
- [ ] Accessibility improvements (WCAG 2.1 AA)
